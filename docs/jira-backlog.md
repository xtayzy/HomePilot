# HomePilot — полный бэклог проекта

Структурированный бэклог всего проекта от начала до конца. Основа: [TECH_SPEC.md](TECH_SPEC.md).

**Актуализация (2026-04-25):** в репозитории появились админ API (`/api/v1/admin`), React-страницы админки (дашборд, пользователи, подписки, визиты с фильтрами по датам — не календарь, исполнители, поддержка, платежи), клиентский раздел поддержки (`/dashboard/support`), forgot/reset password на бэкенде и фронте. Пауза и смена статуса подписки доступны оператору через админку, не из клиентского кабинета. По-прежнему нет CI в `.github`, нет проверки «не позднее чем за 24 ч» и лимита переносов при переносе визита клиентом, нет самообслуживающей паузы/отмены подписки для клиента, нет CRUD тарифов и справочников в админке. Деплой: есть Docker Compose prod и документация (`docs/DEPLOY.md`); готовый прод-стенд в бэклоге не отмечен как завершённый.
